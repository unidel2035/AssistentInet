# Запуск AssistentInet на Windows 10

## Шаг 1 — Установить Python (один раз, ~5 минут)

1. Открой браузер, перейди на https://www.python.org/downloads/
2. Нажми большую кнопку **"Download Python 3.x.x"**
3. Запусти скачанный файл
4. ⚠️ **ВАЖНО:** поставь галочку **"Add Python to PATH"** внизу окна
5. Нажми **"Install Now"**
6. После установки закрой окно

## Шаг 2 — Скачать проект

Скачай архив проекта с GitHub:
https://github.com/unidel2035/AssistentInet/archive/refs/heads/master.zip

Распакуй в удобное место, например `C:\AssistentInet\`

## Шаг 3 — Тест без прибора (проверить что всё работает)

1. Открой папку `C:\AssistentInet\`
2. Дважды кликни **`start.bat`**
3. Выбери режим — напечатай `demo` и Enter (или перетащи `start.bat` в командную строку с параметром `demo`)

Или через командную строку:
```
cd C:\AssistentInet
start.bat demo
```

Открой браузер: **http://localhost:8090** — должен появиться интерфейс с графиком.

## Шаг 4 — Подключить реальный прибор

1. Подключи USB кабель к шумомеру
2. Нажми **Win+X** → **Диспетчер устройств**
3. Найди раздел **"Порты (COM и LPT)"**
4. Запомни номер порта (например `COM3`)
5. Запусти:
```
start.bat COM3
```

Открой браузер: **http://localhost:8090**

## Если что-то пошло не так

| Проблема | Решение |
|---|---|
| "Python не найден" | Переустанови Python с галочкой "Add to PATH" |
| Прибор не отвечает | Проверь кабель, попробуй другой COM порт |
| Порт занят | Закрой другие программы работающие с COM портом |
| В диспетчере нет COM порта | Нужен драйвер USB-UART (CP2102 или CH340) |
